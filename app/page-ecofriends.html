Вход/Регистрация

Проведение, через систему наставничества, экоуроков для учащихся образовательных учреждений и внедрение раздельного сбора отходов в учебных заведениях.

ТЕХНОЛОГИЯ ПРОЕКТА
Применение в системе экопросвещения образовательной программы для школ, включающей комплекс экоуроков, программы наставничества, наглядных пособий и практического руководства для школьников по усовершен- ствованному сбору отходов.
Заключение договоров между региональными операторами на предмет установки в шаговой доступности контейнеров для раздельного сбора отходов, вывоза накопленного вторсырья, фиксации результатов.
Популяризация идеи, путем проведения дополнительных мероприятий: экоквестов, соревнований, акции, экскурсий на предприятия по сортировке и переработке ТКО и пр.
Привлечение спонсоров для поощрения участников проекта призами полезными для школ и их учеников.
Система организации проекта
ОБЪЕДИНЕНИЕ Организаторы объединяются с общественными, коммерческими и государственными организациями
ОБУЧЕНИЕ Организация экоуроков старшеклассниками для учащихся младших классов и проведение квестов
СБОР Внедрение в школе раздельного сбора отходов
НАГРАДА За проведение социально значимого мероприятия, партнеры проекта поощряют участников
Достижения проекта
1.
Проект прошел программу акселерации Moscow Eco Challenge в Москве
2.
Экоуроки были проведены для учащихся школ Саратовской и Астраханской областей
3.
Проект уже реализуется в Саратовской области совместно с Региональным оператором по обращению с ТКО - АО «Управление отходами»
